# 웹 대시보드 (프론트엔드)

주식 스코어링·백테스트 백엔드를 위한 화면. **Next.js(프론트) + FastAPI(백엔드 API)** 2개를 같이 띄웁니다.

## 한 번만: 설치

```bash
# 1) 백엔드 API 의존성 (프로젝트 루트에서)
pip install -e ".[api]"      # fastapi, uvicorn 설치

# 2) 프론트 의존성 (web 폴더에서)
cd web
npm install
```

## 실행 (터미널 2개)

**터미널 A — API 서버 (프로젝트 루트):**
```bash
uvicorn api.main:app --reload --port 8011
```

**터미널 B — 프론트 (web 폴더):**
```bash
npm run dev
```

브라우저에서 http://localhost:3000 접속.

## 데이터 모드

- **실데이터(live):** yfinance로 진짜 S&P500 데이터를 받아 돕니다. 인터넷 필요.
  데이터를 못 받으면 화면에 **에러를 그대로 표시**합니다 (가짜로 성공한 척 안 함).
- **데모(demo):** 인터넷 없이 합성 데이터로 화면만 확인. 결과에 **"⚠️ 데모 데이터"**
  배너가 항상 뜨며, 숫자는 아무 의미 없습니다.

> 처음 실데이터로 돌리면 종목 수만큼 다운로드가 느립니다(수 분). 받은 데이터는
> `data/cache/`에 저장돼 다음부터 빨라집니다. "유니버스 크기"를 줄이면 빠르지만
> 실제 S&P500이 아니니, 진짜 결과는 "전체"로 두세요.

## API 주소 바꾸기

기본은 `http://127.0.0.1:8011`. 다르면 `web/.env.local`에:
```
NEXT_PUBLIC_API_BASE=http://127.0.0.1:8011
```

## 화면

- **오늘의 포트폴리오** — 상위 종목 랭킹·비중, 섹터 분포.
- **백테스트** — 자산곡선 vs SPY, 지표, **합격/불합격 게이트**, 비용 투명화, 생존편향 경고.
